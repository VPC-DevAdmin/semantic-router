The hardware

One server orchestrates the whole stack.

The router runs on a Dell PowerEdge R770 that sits between your users and every inference backend, classifying each prompt and dispatching it to the right place: on-prem CPU, on-prem GPU, or the cloud.

Add a Dell PowerEdge R770 as your routing layer.

One R770 sits at the front of your AI stack as a dedicated orchestration and routing tier. It classifies every prompt in milliseconds and sends it to the cheapest backend that can answer it well, keeping GPUs and frontier APIs for only the queries that truly need them.

Intel Xeon 6 with Intel® AMX

Advanced Matrix Extensions run the routing classifier and small-model inference directly on the CPU. No GPU required for the routing layer itself or for quick answers needing a smaller model.

Orchestration & routing

One server classifies and dispatches every query across on-prem CPU, on-prem GPU, and cloud backends becoming the control plane for your whole model fleet.

Built for concurrency

High core counts, large memory capacity, and fast memory bandwidth keep classification quick at thousands of prompts per second, with PCIe Gen5 to feed local GPUs and NICs.

Keep sensitive data on-prem

Answer private prompts locally and escalate only the hard queries beyond your network. Routing becomes a data-governance boundary, not just a cost lever.
